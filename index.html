Sobre nós:

O que é o JSManager?

O JSManager é um modelo de servidor local gratuito para pequenas empresas de até 25 usuários, podendo ser utilizado como base para um futuro upgrade de servidor, possibilitando gerenciar os usuários locais.

Por que escolher o JSManager?

Fácil implementação

Com o JSManager a implementação de um servidor na sua empresa se torna muito mais fácil e viável.

Gratuito

O linux é um software com licença gratuita, sendo assim mais acessível para todas as pequenas empresas que quererm começar no mercado. O nosso servidor utiliza o UBUNTU SERVER.
